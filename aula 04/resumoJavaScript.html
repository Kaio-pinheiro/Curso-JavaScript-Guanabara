OPERADORES RELACIONAIS

PARA TODA EXPRESSÃO QUE TENHA UM OPERADOR RELACIONAL LIGADO A ELA O RESULTADO DESSA EXPRESSÃO VAI SER SEMPRE UM VALOR BOOLEANO UM VALOR VERDAEIRO OU FALSO

5 > 2 true

7 < 4 false

8 >= 8 true

9 <= 7 false

5 == 5 true O OPERADOR DE IGUALDADE DO JS É ==

4 != 4 false QUER DIZER NÃO IGUAL !=

Se tiver operadores aritméticos e operadores relacionais na expressão, resolver primeiro o aritmético depois os relacionais!!!

Exemplo: var a = 10 e var b = 20

a <= b - 10(Resolve primeiro b - 10 depois a <= b-10 R:true

OPERADOR RELACIONAIS DE IDENTIDADE

5 == 5 true

5 == '5' true

Pro js o sinal de igualdade nao testa o tipo, o valor '5' em 'string' é o mesmo valor

5 === '5' false

Pra saber se são idênticos o mesmo valor e o mesmo tipo, por isso false pois um é number outro string

Alguns exemplos:

preco >= 200.50 // o preço é maior ou igual a 200.50?

idade < 18 //a idade é menor do que 18?

curso == 'Javascript' // o curso é javascript?

n1 != n2 //o primeiro número é diferente do segundo?
