JAVASCRIPT

HTML: conteúdo, é o body

CSS: estilo, é a head

JS: interação, é o script

Para abrir, file-open folder-new folder-new file---ex001.html

Para começar digitar html:5

html lang= "pt-br" para mudar para portugues

Comandos da head

head: cabeça

title: é nome que aparece na aba do chrome

style: pra mecher nas cores e tamanhos da letra, ta dentro é CSS, todo final tem ; ponto e virgula ;

{background-color: cor do fundo

color: cor da letra

font}: tamanho da letra e fonte ex font: normal 20pt ARIAL

h1 {color:}: cor do titulo

p1 {color}: cor do p1

Comandos do body

h1: titulo

p: novo parágrafo

p1: para dividir o texto e poder trocar de cor

Comandos do script

Geralmente o script vai no final do body, para ser carregado depois do codigo base

window.alert('MENSAGEM') ou só alert('MENSAGEM'): alerta com mensagem simples, só opção 'ok' ou 'cancel'

window.confirm('MENSAGEM'): janela de confirmação

window.prompt('MENSAGEM'): janela de resposta

Para saber as respostas precisa de variáveis na frente

Só funciona ESSES comentário dentro do script, fora são outros

______________________________________________________________

//: comentário em uma única linha

/* */: comentário mais de uma linha de código

______________________________________________________________

IDENTIFICADORES:

Podem começar com letra, $ ou _

Não podem começar com números

Pode usar palavras com acento

Não pode conter espaço, se usa anderlaine

Não pode usar palavras do propio java

TIPOS DE DADOS

number Infinity NaN string boolean null undefined object array function

typeof para saber qual o tipo

EXEMPLOS DE typeof

typeof 6 'number'

typeof k 'string'

typeof "6" 'string'

typeof [] 'object'

typeof {} 'object'

typeof NaN 'number'

Se botar dentro de "Aspas" vira 'string'

Se botar dentro de [colchetes] ou {chaves} é 'object'
